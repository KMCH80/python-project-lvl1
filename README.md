# Welcome to Brain Games!      

## Quickstart
```bash
pip install brain-games
```

## Running commands
```bash
1. brain-even
2. brain-calc
3. brain-prime
4. brain-gcd
5. brain-progression 
```

```bash
1. There are 5 games for your brain.
2. Each game has 3 rounds.

```
